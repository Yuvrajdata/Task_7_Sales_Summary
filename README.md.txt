# Sales Summary using SQLite and Python

## About
I created this project using Python, SQLite, Pandas, and Matplotlib. I connected Python to a SQLite database, stored sample sales data, ran SQL queries to calculate total quantity and revenue, and displayed the results in a bar chart.

## Tools Used
- Python
- SQLite
- Pandas
- Matplotlib
- Jupyter Notebook

## Project Files
- Task_7.ipynb
- sales_data.db
- sales_chart.png

## Result
- Connected Python with SQLite
- Calculated total quantity sold
- Calculated product revenue
- Created a revenue bar chart
